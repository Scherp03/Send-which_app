## Server build Setup

Enter the back-end directory and run the following commands

```bash
# Install Dependencies
npm install
# Install Dependencies For Development
npm install --include=dev

# Run Server Initializing Database
npm run dev:init
# Run Server
npm run dev

# Run Tests
npm run test
```
